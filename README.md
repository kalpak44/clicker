# HtmlUnit tests from web interface with scheduling tasks

## Stack of using technologies
Spring Security, Spring Boot, Spring Data JPA, HSQL, JSP, HtmlUnit


## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven
- JSP
- HSQL

## Run
```mvn clean spring-boot:run```
